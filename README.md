Sistema de Avaliação de Artigos Acadêmicos desenvolvido em linguagem C como aplicação prática dos conceitos de Estrutura de Dados.

O sistema automatiza o cadastro de avaliadores, artigos e notas, garantindo organização, confiabilidade e rastreabilidade no processo de avaliação acadêmica. Permite operações como inserção, busca e listagem de dados.

O projeto foi apresentado no Encontro de Engenharia da FUCAPI (ENCOENG).
